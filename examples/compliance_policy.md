# Example compliance source

Policy: Vendor Data Access

Section 1. External vendors may access customer data only after the security team approves the request.

Section 2. The requester must attach the contract, data processing agreement, and access expiration date.

Section 3. Access logs must be retained for three years and reviewed every quarter by the security team.

Ontology extraction target:

- Policy: Vendor Data Access
- Section: Section 1, Section 2, Section 3
- Rule: security approval required, evidence attachment required, log retention required
- Team: security team
- Evidence: contract, data processing agreement, access expiration date, access logs
